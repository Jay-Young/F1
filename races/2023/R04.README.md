# 阿塞拜疆大奖赛

2023 年 4 月 28 日 — 30 日

## 简介

阿塞拜疆大奖赛（阿塞拜疆语：Azərbaycan Qran Prisi）是一场自 2017 年开始举办的一级方程式赛车分站赛事。这场大奖赛举办于阿塞拜疆首都巴库的巴库市街赛道，这条赛道的首场一级方程式赛车赛事为 2016 年的欧洲大奖赛。[^1]
